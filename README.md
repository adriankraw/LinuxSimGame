# LinuxSimGame

## Features

- creating terminal
- closing terminal
- adding Line

## KeyBindings

| Key                 | Description       |
| ------------------- | ----------------- |
| Enter               | adding line       |
| Alt                 | supermod          |
| Alt + Enter         | create terminal   |
| Alt + Shift + Enter | close terminal    |
| Alt + H             | align: horizontal |
| Alt + V             | align: vertical   |
